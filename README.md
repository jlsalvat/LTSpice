# LTSpice
libraries,example and howto (in french) for ltSpice
To install this addon on LTSpice :
1/ install LTSpice http://www.linear.com/designtools/software/#LTspice
2/ download the zip folder on this git and copy to installed LTSpice install folder (C:\Program Files (x86)\LTC)
3/ unzip folder ltspice.zip, this will copy folder LTSpiceIV from github in your local folder LTSpiceIV
4/ verify that in your folder C:\Program Files (x86)\LTC\LTSpiceIV\lib you have a folder named addon (new components)
5/ verify that in your folder C:\Program Files (x86)\LTC\LTSpiceIV\Examples you have a folder named addon (new examples for yours new components)
6/ Launch LTSpice and open an Example on C:\Program Files (x86)\LTC\LTSpiceIV\Examples and test...
7/ enjoy
